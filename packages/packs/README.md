# @agent-pack/packs

Official Agent-Pack modules and loadouts used by the CLI.

## Install

```sh
npm install @agent-pack/packs
```

## Usage

This package is consumed by the CLI. You typically do not need to use it
directly unless you are building tooling around Agent-Pack packs.
